# Design Style Vocabulary

Use these terms to classify analyzed designs in the `meta.style` array.

## Visual Styles

| Category | Terms |
|----------|-------|
| **Modern/Clean** | minimalist, flat, material-design, swiss-style, scandinavian |
| **Depth & Texture** | glassmorphism, neumorphism, skeuomorphic, claymorphism, frosted-glass |
| **Bold & Experimental** | neobrutalism, brutalist, cyberpunk, vaporwave, retro-y2k, memphis |
| **Elegant** | editorial, magazine-style, luxury, art-deco, sophisticated |
| **Playful** | illustrated, cartoon, rounded-bubbly, colorful, whimsical |
| **Data-focused** | dashboard, analytics-first, information-dense, terminal-cli |
| **Typography-first** | big-type, kinetic-typography, high-contrast-type, wide-tracking |
| **3D & Effects** | 3d-icons, 3d-clay, soft-3d, isometric, gradient-illustration |
| **Vintage** | retro-ui, film-grain, paper-texture, halftone, crt-scanlines |
| **Nature & Organic** | earthy, soft-gradients, pastel, warm-neutrals, botanical, wellness |
| **Fintech & Premium** | premium-fintech, institutional, black-gold, subtle-shine, minimal-borders |

## Icon Styles

| Style | Characteristics |
|-------|-----------------|
| `outline` | Thin strokes, no fill, modern |
| `filled` | Solid shapes, bold, clear at small sizes |
| `duotone` | Two-tone fill, depth |
| `hand-drawn` | Sketchy, playful |
| `3d` | Dimensional, glossy |

## Shadow Patterns

| Pattern | When to Use |
|---------|-------------|
| `none` | Flat design, brutalist |
| `subtle` | Minimal depth, clean interfaces |
| `soft` | Neumorphism, gentle elevation |
| `dramatic` | Dark themes, card emphasis |
| `layered` | Multiple shadows for realistic depth |
| `colored` | Shadows matching accent colors |
